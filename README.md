# Anime DataBase
Site simples para ver animes em lançamento usando dados do MyAnimeList.

## Tecnologias
- React
- TypeScript
- Bootstrap
- API Jikan (MyAnimeList)

## Como usar
1. `npm install`
2. `npm start`

## Dados
Usa a API Jikan: `https://api.jikan.moe/v4/`
Jikan é uma API não oficial e de código aberto para `MyAnimeList.net`.
